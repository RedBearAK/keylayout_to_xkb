## Dead keys


Each dead key composes with the next key pressed. A base key with no listed composition falls back to the dead key's accent character followed by the base character (for example, umlaut then `s` gives `¨s`).


### Telugu Sign Virama (reached by D)


Standalone accent: `్` (TELUGU SIGN VIRAMA). 1 compositions.


| Base | Result | Code point | Description |
